# Vendor-based Platform with Platform Orchestrator

## PocketIDP Overview

PocketIDP demonstrates an IDP architecture centered around a **Platform Orchestrator (Humanitec)**, representing a **graph-based backend**.

## Key Components

- **Git**: Version control
- **Backstage**: Developer interface
- **Humanitec Pipelines**: Continuous Delivery
- **Score**: Workload specifications

## Key Characteristic

PocketIDP **explicitly delineates responsibilities** between development and operations teams through its orchestrator-driven architecture.

## Architecture Benefits

- Unified workflow connecting developer code with resource definitions
- More structured approach to platform management
- Clear separation of concerns

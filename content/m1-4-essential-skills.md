# Essential Skills to Become a Platform Engineer

## Technical Skills

- **Infrastructure as Code**: Terraform, Pulumi, Crossplane
- **Container orchestration**: Kubernetes, Docker
- **CI/CD pipelines**: GitHub Actions, ArgoCD, Jenkins
- **Cloud platforms**: AWS, Azure, GCP
- **Programming**: Go, Python, TypeScript

## Product Skills

- **User research**: Understanding developer pain points
- **Product management**: Roadmapping, prioritization
- **Documentation**: Clear, user-friendly docs
- **Metrics and measurement**: DORA metrics, adoption tracking

## Soft Skills

- **Communication**: Explaining technical concepts to varied audiences
- **Empathy**: Understanding developer frustrations
- **Collaboration**: Working across teams
- **Advocacy**: Selling the platform internally

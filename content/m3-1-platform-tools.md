# What Tools Platform Teams Are Using

## Developer Control Plane

- **Version control**: GitHub, GitLab, Bitbucket
- **Developer portals**: Backstage (OSS), Port, Cortex
- **Infrastructure as Code**: Terraform, Crossplane
- **Workload specifications**: Score

## Integration & Delivery Plane

- **CI/CD**: GitHub Actions, ArgoCD, Jenkins, GitLab CI
- **Container registries**: Docker Registry, Harbor
- **Platform Orchestrators**: Humanitec, Kratix, KusionStack

## Resource Plane

- **Infrastructure provisioning**: Terraform, AWS, Azure, GCP
- **Container orchestration**: Kubernetes (EKS, GKE, AKS)
- **Resource management**: Abstract infrastructure from developers

## Monitoring & Logging Plane

- Observability tools for performance insights
- Logging aggregation and analysis
- Essential for mature platform design
